# Guide

The guide exists so wikilinks, Markdown links, backlinks, and embeds have a real target.

## Install

Follow these steps:

1. Open the fixture PR in Commentary.
2. Switch between rendered, diff, and raw modes.
3. Confirm anchors stay attached to the same semantic blocks.
